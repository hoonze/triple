## 어플리케이션 실행 방법

### 1. git clone

``` bash
$ git clone https://github.com/hoonze/triple.git
```

### 2. triple/backend 폴더 접근

```bash
$ cd triple/backend
```

### 3. 프로젝트 빌드

``` bash
$ ./gradlew bootjar
```

### 4. docker 컨테이너 실행

```bash
$ docker-compose up
```
